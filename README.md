# Nails — Gestión de Salón

Aplicación web para la gestión de un salón de uñas. Permite administrar citas, servicios, clientes e inventario desde una interfaz moderna y responsiva.

## Stack

Next.js 15 · TypeScript · Tailwind CSS · App Router

## Features

- Gestión de citas y agenda
- Catálogo de servicios con precios
- Administración de clientes
- Control de inventario de productos
- Interfaz responsiva optimizada para móvil y desktop

## Setup

```bash
npm install
npm run dev
```

Abre `http://localhost:3000` en tu navegador.

```bash
npm run build    # Build para producción
npm start        # Servidor de producción
npm run lint     # Linter
```
